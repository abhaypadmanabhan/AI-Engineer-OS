## Why structural chunking is mandatory on 10-Ks

10-Ks have explicit section anchors:

```
## ITEM 1. BUSINESS
...company description...
## ITEM 1A. RISK FACTORS
...risk disclosures...
## ITEM 7. MANAGEMENT'S DISCUSSION AND ANALYSIS
...MD&A...
## ITEM 8. FINANCIAL STATEMENTS AND SUPPLEMENTARY DATA
...statements + footnotes...
```

The fetcher inserts these markers; the chunker breaks on them. A recursive splitter (LangChain default, LlamaIndex `SentenceSplitter`) crosses these headers and produces chunks that mix Item 1 (descriptive) with Item 1A (risk-flavored) — the generator then mis-attributes risk language to the business overview. Structural chunking eliminates the entire failure class.

```python file=chunk.py
def chunk_filing(text, ticker, seed=0):
    sections = split_on(r"^## (ITEM \d+[A-Z]?\.\s+[^\n]+)$", text)
    chunks = []
    for section, body in sections:
        if len(body) <= 1500:
            chunks.append({"ticker": ticker, "section": section, "text": body})
        else:
            # Long sections (Risk Factors, MD&A) need secondary splits;
            # seed-jittered boundary surfaces real chunking variance.
            chunks.extend(_split_long(body, ticker, section, seed))
    return chunks
```

## Why two seeds, and what they actually control

Production RAG is not bit-deterministic. Server-side LLM variance, judge sampling, and tie-broken sort orders all introduce noise at every component boundary. A single-seed run hides this. Two seeds with explicit jitter on the chunk boundary expose it.

```python file=seeds.py
SEEDS = (7, 42)
# Each seed run produces (faith, ctx_prec, ans_rel, ctx_rec).
# Report: mean ± stdev across seeds.
# Gate: mean >= threshold.   stdev tells you margin of safety.
```

If faithfulness lands 0.91 ± 0.02 we ship; if 0.85 ± 0.05 we are at the edge and need to fix before scaling. The number that matters is not the mean alone — it is the lower confidence bound.

## Routing — three transforms, one pipeline

```python file=route.py
def route(q, qtype):
    if qtype == "comparison":  return _step_back(q)   # 3.7 — abstraction surfaces both filings
    if qtype == "factual":     return [q]             # baseline — literal tokens win
    return _hyde(q)                                   # multi_hop — answer-shaped retrieval
```

Every transform from 3.7 has a place; none is universal. The router is one Sonnet call cheaper than running all transforms in parallel and ~95% as accurate on the labeled dataset.

## Faithfulness ≥ 0.85 AND context_precision ≥ 0.75

Both must pass. Faithfulness alone is gameable (the generator picks one true sentence and stops); context_precision alone does not enforce grounded generation. The conjunction is the contract. Per SPEC.md L3 gate.

```python file=gate.py
passed = (
    summary.faithfulness_mean      >= 0.85 and
    summary.context_precision_mean >= 0.75
)
```
