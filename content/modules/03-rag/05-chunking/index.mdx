Four chunkers. One corpus. Same 20-Q eval set. Pick the winner.

```python file=fixed.py
def fixed(text: str, size: int = 500, overlap: int = 50) -> list[str]:
    step = size - overlap
    return [text[i:i+size] for i in range(0, len(text), step)]
```

Fixed is the baseline. It is also what you saw failing in lesson 3.3 — boundary splits everywhere.

## Recursive — hierarchy of separators

```python file=recursive.py
SEPARATORS = ["\n\n", "\n", ". ", " ", ""]

def recursive(text: str, size: int = 500) -> list[str]:
    if len(text) <= size:
        return [text]
    for sep in SEPARATORS:
        if sep in text:
            parts, buf = [], ""
            for piece in text.split(sep):
                add = piece + (sep if sep else "")
                if len(buf) + len(add) > size and buf:
                    parts.append(buf)
                    buf = add
                else:
                    buf += add
            if buf:
                parts.append(buf)
            # recurse on oversized pieces
            out: list[str] = []
            for p in parts:
                out.extend(recursive(p, size) if len(p) > size else [p])
            return out
    return [text[i:i+size] for i in range(0, len(text), size)]
```

Splits on paragraph breaks first, then line breaks, then sentences, then words. Cuts land at natural boundaries. This is what LlamaIndex's `SentenceSplitter` and LangChain's `RecursiveCharacterTextSplitter` do under the hood.

## Semantic — embed, detect breakpoints

```python file=semantic.py
import numpy as np, voyageai
vo = voyageai.Client()

def semantic(text: str, size: int = 500, threshold_pct: int = 90) -> list[str]:
    sents = [s.strip() for s in text.split(". ") if s.strip()]
    if len(sents) < 3:
        return [text]
    E = np.array(vo.embed(sents, model="voyage-3", input_type="document").embeddings)
    E /= np.linalg.norm(E, axis=1, keepdims=True)
    # Adjacent-sentence cosine: low similarity = topic boundary
    sims = (E[:-1] * E[1:]).sum(axis=1)
    cut = np.percentile(sims, 100 - threshold_pct)   # break where similarity is bottom-10%
    chunks, buf = [], sents[0]
    for i, s in enumerate(sents[1:]):
        if sims[i] < cut and len(buf) > 50:
            chunks.append(buf); buf = s
        else:
            buf = f"{buf}. {s}"
    chunks.append(buf)
    return chunks
```

<CostMeter cost="$0.02" latency="0.8s" tokens="0" />

Embeds every sentence once to find topic shifts — paid per corpus, not per query. Wins on narrative text (papers, transcripts). Loses when the cost of embedding during ingest matters more than a 3-5% recall bump.

## Structural — use the document's own structure

```python file=structural.py
import re

def structural_md(text: str, size: int = 1200) -> list[str]:
    """Split markdown at headings; keep heading with its section. Fallback-split oversized sections."""
    parts = re.split(r"(?=^#{1,3} )", text, flags=re.M)
    out: list[str] = []
    for p in parts:
        if len(p) <= size:
            out.append(p.strip())
        else:
            out.extend(recursive(p, size))
    return [o for o in out if o]
```

For code, swap the regex for an AST walk (`ast.parse` in Python, `tree-sitter` for polyglot). For HTML, `BeautifulSoup` + `<section>` / `<h2>`. **Structure is free signal.** Throw it away only if you must.
