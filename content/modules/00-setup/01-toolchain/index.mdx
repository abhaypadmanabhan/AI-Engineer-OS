Install the entire stack in under two minutes.

```bash file=install.sh
# macOS / Linux — one-liner installs uv, which brings its own Python
curl -LsSf https://astral.sh/uv/install.sh | sh

# create a project, pin Python 3.12, install the core SDKs
uv init ai-engineer-warmup && cd ai-engineer-warmup
uv python pin 3.12
uv add anthropic openai httpx pydantic python-dotenv
uv add --dev ruff
```

That is the entire toolchain. No `pip install`, no `python -m venv`, no `pyenv`. `uv` replaces them all and runs 10–100× faster because it's Rust.

## Why this stack

`uv` is the package manager. It resolves dependencies deterministically, writes a lockfile, and installs Python itself — so every teammate gets the exact same environment without fighting `pyenv`.

`ruff` is the linter **and** formatter. Runs in milliseconds on massive repos. Drop-in replaces `black`, `isort`, `flake8`, `pylint`.

`pyproject.toml` is the single source of truth. Every modern tool reads it.

`direnv` auto-loads `.env` when you `cd` into the project — so `os.environ["ANTHROPIC_API_KEY"]` just works without you ever `source`-ing anything.

## Minimum viable `pyproject.toml`

```toml file=pyproject.toml
[project]
name = "ai-engineer-warmup"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = [
  "anthropic>=0.40",
  "openai>=1.60",
  "httpx>=0.27",
  "pydantic>=2.9",
  "python-dotenv>=1.0",
]

[tool.ruff]
line-length = 100
target-version = "py312"

[tool.ruff.lint]
select = ["E", "F", "I", "UP", "B", "SIM"]
```

## direnv in 30 seconds

```bash
brew install direnv               # or: apt install direnv
echo 'eval "$(direnv hook zsh)"' >> ~/.zshrc   # bash: replace zsh

# inside the project
echo 'dotenv' > .envrc
direnv allow
```
